# 客服管理后台 — 产品需求文档

## 一、产品概述

客服管理后台是一个面向内部客服人员的 Web 端管理系统，用于为三个产品（掌上AD、AD搭子、AD晴雨表）的用户创建和管理账号。客服通过飞书登录后台，帮助用户开通账号、设置用量上限、管理邀请码、查看操作日志。

## 二、用户角色

| 角色 | 说明 |
|------|------|
| 客服人员 | 通过飞书登录，执行账号创建、编辑、重置密码、禁用/启用等操作 |

## 三、登录

- 使用飞书 OAuth 登录，仅限内部客服人员
- 登录后顶部导航栏显示当前客服姓名
- 支持退出登录

## 四、功能模块

### 4.1 数据概览

顶部展示三个统计卡片：

| 指标 | 说明 |
|------|------|
| 总账号数 | 系统中所有账号总数 |
| 今日新增 | 当天创建的账号数 |
| 待处理 | 状态异常或需要人工介入的账号数 |

下方展示"最近操作"列表，显示最近 5 条操作记录（操作类型 + 目标账号 + 时间）。

### 4.2 账号管理

#### 4.2.1 账号列表

表格字段：

| 字段 | 说明 |
|------|------|
| 邮箱 | 用户邮箱，作为登录凭证 |
| 产品 | 掌上AD / AD搭子 / AD晴雨表（一个账号只对应一个产品，同一邮箱用多个产品需分别创建） |
| 用量 | 已用/上限，带进度条。不同产品含义不同：掌上AD=广告账号数，AD搭子=模板数，AD晴雨表=消息查看数 |
| 邀请码 | 该用户的唯一邀请码 + 该用户使用的别人的邀请码 |
| 状态 | 正常 / 已禁用 |
| 创建时间 | 账号创建日期 |
| 操作 | 用量、编辑、重置密码、禁用/启用 |

支持功能：
- 按邮箱搜索
- 按产品筛选（掌上AD / AD搭子 / AD晴雨表）
- 按状态筛选（正常 / 已禁用）
- 分页（每页 10 条）

#### 4.2.2 创建账号

表单字段：

| 字段 | 必填 | 说明 |
|------|------|------|
| 邮箱 | 是 | 用户邮箱 |
| 初始密码 | 是 | 支持自动生成（12位随机）+ 手动输入 + 一键复制 |
| 产品 | 是 | 单选下拉：掌上AD / AD搭子 / AD晴雨表 |
| 该用户的邀请码 | 否 | 每人一个，留空自动生成（按产品前缀 ZS-/AD-/QY- + 6位随机码） |
| 使用的邀请码 | 否 | 该用户注册时填写的别人的邀请码，只能填一个 |
| 用量设置 — 已用 | 否 | 默认 0 |
| 用量设置 — 上限 | 否 | 默认 5 |
| 备注 | 否 | 自由文本 |

创建成功后自动复制密码到剪贴板。

#### 4.2.3 编辑账号

- 邮箱不可修改
- 产品可修改
- 可修改：邀请码、使用的邀请码、用量（已用/上限）、备注
- 密码字段留空表示不修改，填写则重置

#### 4.2.4 用量查看与编辑

点击"用量"按钮弹出详情面板：
- 大号数字展示已用/上限 + 百分比进度条
- 颜色：<60% 绿色，60-90% 黄色，≥90% 红色
- 支持点击"编辑"按钮切换为编辑模式，直接修改已用和上限
- 显示最后活跃时间、所属产品、邀请码、使用的邀请码、状态、创建时间、备注

各产品用量含义：

| 产品 | 用量含义 | 单位 |
|------|----------|------|
| 掌上AD | 广告账号数 | 个 |
| AD搭子 | 模板数 | 个 |
| AD晴雨表 | 消息查看数 | 次 |

#### 4.2.5 其他操作

- **重置密码**：二次确认后重置，新密码自动复制到剪贴板
- **禁用/启用**：二次确认后切换状态

### 4.3 操作日志

记录所有客服操作，字段：

| 字段 | 说明 |
|------|------|
| 时间 | 操作时间，精确到分钟 |
| 操作人 | 客服姓名 |
| 操作类型 | 创建账号 / 重置密码 / 禁用账号 / 启用账号 / 修改信息 |
| 详情 | 目标邮箱 + 产品 + 附加信息 |

- 按时间倒序排列
- 分页（每页 10 条）

## 五、邀请码规则

- 每个用户有且只有一个邀请码（myCode），格式为 `{产品前缀}-{6位随机码}`
- 一个邀请码可以被多个用户使用
- 每个用户注册时只能填写一个别人的邀请码（usedCode）
- 产品前缀：掌上AD → ZS，AD搭子 → AD，AD晴雨表 → QY

## 六、技术要求

- 前端：单页应用，响应式适配移动端
- 登录：飞书 OAuth 2.0
- 后端：RESTful API
- 数据库：存储账号、邀请码关系、操作日志
